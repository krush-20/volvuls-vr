# Volvulus WebXR Scene

This project is a browser-based VR experience built with [A-Frame](https://aframe.io), designed to run on WebXR-compatible devices like the Meta Quest 3.

It features a custom 3D character model ("Volvulus") rendered in a fully immersive virtual environment, accessible via any WebXR-supported browser.
